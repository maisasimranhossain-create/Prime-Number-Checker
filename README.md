# Prime Number Checker in C

A beginner-friendly C program that checks whether a number is prime or not.

## Features
- Takes user input
- Checks if the number is prime
- Simple logic using loops and if-else
- Written in C

## How to Run

Compile:

```bash
gcc prime.c -o prime
```

Run:

```bash
./prime
```
